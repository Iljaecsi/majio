# 🏠 Majio v0.03

**Умный учёт в вашем доме** — простое веб-приложение для подачи показаний 5 счетчиков.

## 🌐 Демо
[https://iljaecsi.github.io/majio/](https://iljaecsi.github.io/majio/)

## ✨ Возможности
- 📝 Подача показаний 5 счетчиков:
  - ⚡ Электричество (дневной)
  - ⚡ Электричество (льготный)
  - 💧 Холодная вода
  - 🔥 Горячая вода
  - 🔥 Газ
- 📊 История всех показаний
- 🗑️ Удаление записей
- 📈 Статистика потребления
- 👤 Профиль пользователя
- 🔐 Регистрация и вход
- 🌓 Переключение светлой/тёмной темы
- 🌍 Переключение языка (Русский/Эстонский)
- 💾 Автосохранение в localStorage
- 📱 Полная адаптивность (ПК, планшеты, смартф

## 🛠️ Технологии
- HTML5
- CSS3 (Glass-morphism, анимации)
- JavaScript (ES6+)
- Font Awesome 6
- LocalStorage

## 📱 Поддержка
- Windows 10/11
- macOS
- Linux
- iOS
- Android
