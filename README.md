# 🏠 Majio

**Tark arvestus sinu kodus** — простое и удобное веб-приложение для подачи показаний счетчиков.

## ✨ Возможности
- 📝 Подача показаний (электричество, холодная/горячая вода)
- 📊 История всех показаний в таблице
- 🗑️ Удаление отдельных записей
- 📥 Экспорт в CSV
- 💾 Автосохранение в localStorage
- 📱 Полная адаптивность (ПК, планшеты, смартфоны)


## 🛠️ Технологии
- HTML5
- CSS3 (Glass-morphism, анимации)
- JavaScript (ES6+)
- LocalStorage

## 📱 Поддерживаемые устройства
- 💻 Компьютеры (Windows, macOS, Linux)
- 📱 Смартфоны (iOS, Android)
- 📟 Планшеты
- 📺 Любые устройства с браузером
